# 3. CSS Behind the Scenes

## Three Pillars of Writing Good HTML & CSS

1. Responsive Design

   - fluid webpage layouts
   - media queries
   - responsive images
   - correct units
   - mobile first

2. Maintainable & Scalable Code

   - clean
   - easy to read
   - growth
   - reusable
   - organized files
   - class names (BEM)
   - organized HTML structure

3. Performance
   - less HTTP requests
   - less code
   - compress code
   - CSS preprocessor (SASS)
   - less images
   - compressed images
